# Tugas Besar Reconnaissance (Pengintaian Pasif dan Aktif)

Repositori ini berisi seluruh artefak tugas besar mata kuliah **Ethical Hacking / Keamanan Jaringan** tentang pengintaian pasif dan aktif terhadap target di jaringan lokal.

# Dokumentasi 
  Berisi dokumentasi teknis:
  - Screenshot hasil perintah **Nmap** (host discovery, TCP SYN scan, UDP scan, service & version detection, OS fingerprinting).
  - Screenshot **Wireshark** yang menangkap trafik TCP/UDP, ARP, DNS/MDNS/LLMNR, DHCP, dll.
  - File capture Wireshark (`*.pcapng`) jika diperlukan.
